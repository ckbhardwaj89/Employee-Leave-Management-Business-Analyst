# Employee Leave Management System (Business Analyst Project)

## Project Overview
This is a Business Analyst portfolio project for an Employee Leave Management System (ELMS).

The project demonstrates the complete Business Analysis lifecycle including:
- Business Requirements
- Functional Requirements
- User Stories
- Wireframes
- Jira Sprint Planning
- Confluence Documentation

---

## Business Goal

The objective is to digitize the employee leave process by allowing employees to:

- Login securely
- View leave balance
- Apply for leave
- Track leave status
- Managers approve/reject requests
- HR generate reports

## Sample User Story

**As an employee, I want to apply for leave online so that I can request leave without sending emails.**

Acceptance Criteria

- Employee is logged in
- Employee selects leave type
- Employee selects dates
- Employee enters reason
- Leave balance is validated
- Request is submitted
- Manager receives notification
