# Employee Leave Management System (Business Analyst Project)

## Project Overview
This project demonstrates the complete Business Analysis lifecycle for an Employee Leave Management System using Agile Scrum methodology. The project was created to showcase practical Business Analyst skills including requirement gathering, documentation, user story creation, backlog management, sprint planning, and wireframing using Jira and Confluence.

---

## Business Objectives

- Automate employee leave requests
- Streamline manager approval process
- Maintain leave balance
- Generate leave reports
- Improve HR process efficiency
